# service-starter

Backend / API service skeleton: entrypoint, a health/readiness probe, and per-language command wiring for the v1.26 merge gate.

This is a Grimoire quick-start scaffold for the `service` profile. Replace this README with your project's. The chosen language variant's commands (test/build/lint/typecheck) are wired by `grm-quick-start-template` on apply. See `docs/design/defaults-quickstart-design.md`.
