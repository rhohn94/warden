# lib-starter

Reusable library skeleton: a public API module, a test harness (framework-required baseline for the library shape), and packaging wiring.

This is a Grimoire quick-start scaffold for the `lib` profile. Replace this README with your project's. The chosen language variant's commands (test/build/lint/typecheck) are wired by `grm-quick-start-template` on apply. See `docs/design/defaults-quickstart-design.md`.
