# Template files

> **Up:** [↑ Docs](../README.md)

Static, **copy-to-start** templates — blank starting points you duplicate when
creating a new artifact, then fill in. This tier holds templates **only**.

**Not** for operational/live files. Documents that are read and edited
frequently as part of normal work — `roadmap.md`, `version-history.md`, the
active `release-planning/` plan — are operational state, not templates, and live
in their own tiers.

## Contents

- [`design-doc-template.md`](design-doc-template.md) — the house-layout skeleton
  for a new `docs/design/{feature}-design.md`. Mirrors what the
  `grm-design-doc-scaffold` skill emits; copy it when starting a design doc by hand.
