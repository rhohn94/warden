> one section per tagged version, bullets aimed at users. The complete internal

## v1.3.4 — Reliable releases

- Warden releases now come with downloadable builds: every GitHub Release
  ships a ready-to-run macOS binary, a manifest, and checksums so you can
  verify what you downloaded.
- Refreshed development tooling and an updated design-token set under the
  hood — nothing changes in how Warden looks or behaves day to day.
